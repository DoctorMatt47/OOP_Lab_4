Detailed Description

template<class T>
class ArrayList< T >

List based on array.

Container with data, which is representing array that can change in size. ArrayList is very efficient accessing its elements and relatively efficient adding or removing elements from its end. This array may need to be reallocated in order to grow in size when new elements are inserted, which implies allocating a new array and moving all elements to it.
