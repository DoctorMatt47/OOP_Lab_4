Detailed Description

template<class T>
class LinkedList< T >

Doubly-linked list.

LinkedList is a container that supports constant time insertion and removal of elements from anywhere in the container. It is implemented as a doubly-linked list. This container provides bidirectional iteration capability. Cannot be inherited.

Template Parameters
TThe type of the elements.
